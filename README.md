# Forklift

AI listing automation for eBay sellers.

Forklift helps sellers turn product photos into ready-to-post listings with AI-assisted workflows. Upload photos, let the AI generate listing drafts, review them, and publish faster than with manual listing processes.

## Overview

Forklift is built for sellers who want to reduce the time spent on copywriting, form filling, and repetitive listing work. The platform is designed to help users go from raw photos to live marketplace listings with more speed, consistency, and control.

## Core Value Proposition

- Generate complete listing drafts from product photos
- Speed up listing creation with AI-assisted workflows
- Work on multiple listings in parallel
- Reduce manual effort for titles, descriptions, item specifics, categories, and pricing
- Customize whole process with your own prompts
- Use a dedicated dashboard for managing listing activity

## Key Features

### 1. AI Draft Generation
Upload product photos and optional notes, and Forklift generates a complete listing draft. This includes:

- Title
- Description
- Category
- Item specifics
- Pricing suggestions

### 2. Parallel Listing Creation
Create and edit several listings side by side, then submit them together. This is especially useful for sellers managing larger inventories or batch uploads.

### 3. Faster Listing Workflow
Forklift is positioned as a faster alternative to manual listing creation, helping sellers move from photos to live listings in fewer steps.

### 4. Seller Dashboard
Users get access to a Forklift dashboard to manage and work with AI-generated listings.

## Workflow

A simplified workflow looks like this:

1. Upload photos
2. Generate AI draft
3. Review and edit details
4. Create listing
5. Publish to marketplace

## Supported Marketplace

Forklift currently presents itself as built for **eBay** sellers.
